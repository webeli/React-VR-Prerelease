Getting Started #

Setting Up React VR #

Before starting your first React VR project, you'll need to install the dependencies used to build and manage React VR apps: Node.js, and the React VR CLI. If you already have Node installed, make sure it is at least version 4.0.

  • Mac: On Mac, we recommend installing Node with Homebrew.

  • Windows: You can grab a Windows installer from Node's downloads page.

  • Linux: Visit this web page to find specific instructions for your distro's package manager.

With Node installed, you can install all the other prerequisites for both the Samples and creating your own project.

Running the Samples #

To run the sample applications you must first install the required packages. To do this enter the navigate to the root folder of the package on your file system and run

npm install

This will install the required node_modules, including those of React VR, once it has completed the download you can start the server to view the content.

npm start

will do this for you.

At this point your browser here for a convenient jump point.

Once you have finished exploring the samples it is time to create your first project. First stop your packager and then navigate to a folder outside of preview under which you will place your project.

Creating your first project #

Use the react-vr-cli to generate a new project called “WelcomeToVR.” The tool will create a new folder containing the scaffolding for a React VR project, and it will fetch all of the necessary external dependencies. Once setup has completed, run npm start to initialize your local development server and open the web page for your new project.

First you must install the react-vr-cli

npm install -g react-vr-cli

Once the cli is installed running

react-vr init WelcomeToVR

will create a new application under the WelcomeToVR folder and install the required dependencies.

When the process finishes, you should enter the WelcomeToVR folder and type npm start. Once running open your browser to http://localhost:8081/vr/index.html, and you should see something similar to the following:

Congratulations and welcome to React VR! Click and drag around the world in your browser, or open it up on your mobile device to see more.

If you're using an experimental web browser that supports WebVR, you should be able to enter your headset and explore the world in full VR. Please see VR Browsers page for details on VR capable browsers.
